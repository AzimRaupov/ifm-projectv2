# Ai-PathFinder

**Ai-PathFinder** — интеллектуальная платформа для генерации образовательных курсов в виде дорожных карт. Система подбирает материалы и тесты для каждого шага обучения, отслеживает прогресс пользователя и развитие навыков.

---

## Основные возможности

- **Генерация курсов**: AI строит индивидуальные дорожные карты обучения под конкретные цели пользователя.  
- **Шаги обучения**: Каждый курс делится на шаги с соответствующими материалами и тестами.  
- **Материалы и тесты**: Для каждого шага предоставляются лекции, статьи, видео и тестовые задания для проверки знаний.  
- **Отслеживание прогресса**: Система фиксирует выполнение шагов и прогресс по курсу.  
- **Навыки и достижения**: Ai-PathFinder оценивает рост навыков пользователя и рекомендует дальнейшие шаги.  
- **Адаптивность**: Курсы подстраиваются под скорость обучения и текущий уровень знаний.

---

## Технологии

- Backend: Laravel
- Frontend: Bootstrap 5
- База данных: MySQL  
- AI-модель: Gpt-4o для генерации курсов и тестов  
- Система отслеживания прогресса и навыков: встроенные алгоритмы анализа

## Установка

1. Клонируйте репозиторий:  
```bash
git clone https://github.com/yourusername/Ai-PathFinder.git
cd Ai-PathFinder
php artisan migrate
```
2. Установка зависимости:  
```bash
composer install
```
3.Запуск Процессов
```bash
php artisan queue:work
```
4. Старт:  
```bash
php artisan serve
```
